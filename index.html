How the bot works

Trades 5-minute Polymarket binary markets: BTC / ETH / SOL Up or Down.

1. Quote/Book tick → 2. Market context → 3. EV gate → 4. Risk guards → 5. Entry → 6. Exit policy

Entry logic (EV gate)

  • edge = p_estimated − token_price from calibrated table
  • Time window: 45–180s to expiry (DOWN ≤120s)
  • Filters: spread, depth, book quality, Binance confirm
  • Price caps and MAX_UP_PER_SLOT = 1 per 5m slot
  • Strict selection: ~10–17% of signals pass replay filters

Exit logic

  • take_profit — main profit driver (+$251, 100% WR on TP)
  • trailing_take_profit — lock in on pullback from peak
  • deep_drawdown_cut — stop at deep drawdown (0.26)
  • ev_exit — exit when live model edge turns negative
  • momentum_cut disabled (0% WR historically)

System architecture

Browser → FastAPI → Nautilus Strategy → Polymarket CLOB + Binance sidecar → trade journal
