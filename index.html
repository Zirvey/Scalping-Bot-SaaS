Как работает бот

Торгует 5-минутные бинарные рынки Polymarket: BTC / ETH / SOL Up or Down.

1. Quote/Book tick → 2. Market context → 3. EV-гейт → 4. Risk guards → 5. Entry → 6. Exit policy

Логика входа (EV-гейт)

  • edge = p_estimated − token_price из калиброванной таблицы
  • Окно времени: 45–180s до экспирации (DOWN ≤120s)
  • Фильтры: spread, depth, book quality, Binance confirm
  • Price caps и MAX_UP_PER_SLOT = 1 на 5m-слот
  • Строгий отбор: ~10–17% сигналов проходят replay-фильтры

Логика выхода

  • take_profit — главный драйвер (+$195, 100% WR на TP)
  • trailing_take_profit — фиксация на откате от пика
  • deep_drawdown_cut — стоп при глубокой просадке (0.26)
  • ev_exit — выход при потере edge по live-модели
  • momentum_cut отключён (исторически 0% WR)

Архитектура системы

Browser → FastAPI → Nautilus Strategy → Polymarket CLOB + Binance sidecar → trade journal
